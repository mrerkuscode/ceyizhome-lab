# Final MVP Readiness Plan

## MVP kabul kriterleri

- P0/P1 ürün hatası yok.
- Etiket Studio drag/resize/zoom/font/renk/PDF payload çalışıyor.
- PDF/PNG output validation geçiyor.
- Yazdır direct print yapmıyor; kullanıcı onay modalı açıyor.
- Queue doğru PDF'i alıyor ve missing/stale PDF'i yazdırmıyor.
- Etiket Çıktıları müşteri çıktısını teknik raporlardan ayırıyor.
- Toplu Etiket en az 100 satırı galeri olarak yönetiyor.
- Yeni Model Ekle teknik editör açmadan model oluşturuyor.
- Ayarlar direct print kapalı bilgisini net gösteriyor.
- Yardım sistemi temel kullanıcı akışını anlatıyor.

## Release Candidate için ek kriterler

- Temiz kurulum paketi ve son kullanıcı rehberi doğrulanmış olacak.
- Canonical screenshot seti tek klasörde olacak.
- Eski test verisi müşteri ekranlarını kirletmeyecek.
- RDWorks true outline/offset tamamlanmadıysa açık P1 teknik risk olarak kalacak; üretime hazır iddiası kullanılmayacak.
