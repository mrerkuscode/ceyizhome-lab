# User Quickstart Training Plan

Tarih: 2026-05-13

## Amaç

Bu doküman normal kullanıcı için kısa, teknik olmayan hızlı başlangıç akışını tarif eder.

Hedef kullanıcı CorelDRAW, Illustrator, RDWorks veya teknik dosya mantığını bilmek zorunda değildir. Kullanıcı model seçer, yazıyı düzenler, PDF/PNG oluşturur ve isterse güvenli şekilde yazdırma sırasına ekler.

## 10 Dakikalık Eğitim Akışı

### 1. Programı Aç

Mesaj:

> Cyzella Production Studio etiket üretimi için hazırlanmış güvenli bir üretim ekranıdır. Yazıcı ve lazer otomatik çalışmaz.

Kullanıcıya göster:

- Ana Sayfa
- Etiket Modelleri
- Etiket Studio
- Toplu Etiket
- Yazdırma Sırası
- Etiket Çıktıları

### 2. Model Seç

Akış:

1. `Etiket Modelleri` sayfasına gir.
2. Görselden modeli seç.
3. Hazır/eksik durumunu kontrol et.
4. `Etiket Hazırla` veya `Studio'da Düzenle` butonuna bas.

Kullanıcıya anlat:

> Görsel eksikse model üretime hazır değildir. Önce görsel bağlanmalıdır.

### 3. Etiketi Düzenle

Akış:

1. İsim yazısına tıkla.
2. İsim, tarih ve not alanlarını değiştir.
3. Yazıyı mouse ile taşı.
4. Köşeden veya kenardan boyutlandır.
5. Font preset seç.
6. Renk seç.
7. Gerekirse `Akıllı Düzen` kullan.

Kısa kısayollar:

- Ok tuşları: hassas taşıma
- `Shift + Ok`: daha büyük adım
- `Ctrl + Z`: geri al
- `Ctrl + Y`: ileri al
- `Delete`: seçili ek yazıyı sil

### 4. Çıktı Oluştur

Akış:

1. `PDF/PNG Oluştur` butonuna bas.
2. `PDF'i Gör` ile kontrol et.
3. `PNG Önizle` ile görseli kontrol et.
4. Uygunsa `Yazdır` veya `Yazdırma Sırasına Ekle` seç.

Güvenlik mesajı:

> Yazdır butonu yazıcıyı sessiz çalıştırmaz. PDF açılır veya kullanıcı onayına sunulur.

### 5. Yazdırma Sırası

Akış:

1. `Yazdırma Sırası` sayfasına gir.
2. Sıradaki işleri kontrol et.
3. PDF'i aç.
4. Yazdırma modalındaki bilgileri kontrol et.
5. Kullanıcı yazdırmayı manuel onaylar.
6. İş bittiyse `Yazdırıldı İşaretle`.

### 6. Etiket Çıktılarını Bul

Akış:

1. `Etiket Çıktıları` sayfasına gir.
2. PDF/PNG/Batch kartlarını filtrele.
3. Sağ preview panelinden doğru çıktıyı kontrol et.
4. Gerekirse `Studio'da Aç`, `Tekrar Üret` veya `Sıraya Ekle`.

### 7. Excel'den Toplu Üret

Akış:

1. `Toplu Etiket` sayfasına gir.
2. Örnek Excel'i indir.
3. Excel'i doldur.
4. Excel'i yükle ve kontrol et.
5. Hatalı satırları düzelt.
6. Galeride kartları kontrol et.
7. Gerekirse bir karta tıkla, düzenle, kaydet.
8. Hazır olanları toplu üret.
9. Yazdır veya sıraya ekle.

Kullanıcıya anlat:

> Hatalı satırlar sessizce üretilmez. Önce düzeltilir veya üretimden çıkarılır.

## Hata Durumunda Kullanıcı Ne Yapmalı?

### Görsel Eksik

Çözüm:

- Etiket Modelleri sayfasında modeli seç.
- `Görsel Bağla` butonunu kullan.

### Yazı Sığmıyor

Çözüm:

- Etiket Studio'da `Yazıları Sığdır` kullan.
- Font boyutunu düşür.
- Alanı genişlet.

### PDF Oluşmadı

Çözüm:

- Eksik model/görsel uyarısını kontrol et.
- Tekrar `PDF/PNG Oluştur` dene.
- Hata devam ederse teknik raporu kontrol ettir.

### Yazdırma Sırasına Eklenmedi

Çözüm:

- Önce PDF/PNG oluştur.
- Output validation geçti mi kontrol et.
- Stale/bozuk dosya varsa yeniden üret.

## Kullanıcıya Söylenmeyecek Teknik Terimler

Normal eğitimde şunlar gereksiz:

- JSON
- field id
- backend path
- template path
- CDR/AI path
- debug output
- direct print config

Bu bilgiler sadece Teknik Mod veya teknik rehberde kalmalı.

## Eğitim Videosu İçin Kısa Senaryo

1. Ana Sayfa açılır.
2. Etiket Modelleri'nde hazır model seçilir.
3. Etiket Studio'da isim değiştirilir.
4. Yazı sürüklenir ve büyütülür.
5. Renk ve font değiştirilir.
6. PDF/PNG oluşturulur.
7. Yazdır modalı gösterilir.
8. Yazdırma sırasına eklenir.
9. Etiket Çıktıları'nda çıktı bulunur.
10. Toplu Etiket'te örnek Excel yüklenir ve galeri gösterilir.

## Kabul Kriteri

Bu hızlı başlangıç yeterli sayılırsa:

- Kullanıcı tek etiket üretimini yardım almadan yapabilir.
- Kullanıcı Excel ile toplu üretim mantığını anlar.
- Kullanıcı yazıcının otomatik çalışmayacağını bilir.
- Kullanıcı hata durumunda nereye bakacağını bilir.
- Teknik ekranlara girmeden üretim yapabilir.
